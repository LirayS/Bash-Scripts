# Honeypot-Alerter

#
# INSTALLATION
# git clone https://github.com/LirayS/Bash-Scripts/Honeypot-Alerter.git
# cd Honeytpot-Alerter
# chmod +x *.sh
# sudo ./honeytpot.sh --- in 1st terminal
